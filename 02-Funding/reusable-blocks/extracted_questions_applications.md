## Cross-application observations

- **Word/char limits vary widely:** Google uses strict per-field word counts (75/100/150/200 words); Astera uses character limits (300/1000 chars); YC uses a 50-char tagline and otherwise unconstrained prose; Foresight uses 350-char project summary and otherwise unconstrained; Brains publishes no limits but filled app averages ~150–250 words per Heilmeier answer.
- **Heilmeier-family core (problem · SOTA · approach · impact · risks · cost · time · exams)** is present in some form in **every** application, but the granularity and naming differs. Brains is closest to explicit Heilmeier (6 questions + why-not-academia). Google expands each Heilmeier axis into 2–4 sub-prompts.
- **Extensions beyond Heilmeier appear in every non-YC app:**
  - **Open-source / public goods:** Google Q13/Q24 (required), Astera Q8 (required), Foresight (preferred), Brains (implicit in FRO framing).
  - **Why-not-academia / FRO-fit:** Astera Q6 (explicit), Brains B7 (explicit), YC YC17 (implicit), Foresight (implicit via focus-area fit).
  - **End-user + beneficiary framing:** Google Q18 (required), Astera Q5 (required), Brains B4 (required), YC YC24 (required).
  - **Responsible AI / safety:** Google Q23 (required), Foresight F17 (required, a first-class prompt).
  - **Equity / accessibility:** Google Q18.a (implicit in "populations that benefit"), Foresight F19 (implicit in 3-yr scaling).
  - **Team / founder fit:** Astera Q3 (required), Brains "Program Leadership Readiness" (required), YC founders section (required), NIH-style investigator framing in Foresight F9.
  - **Budget / resourcing depth:** Google has 3–5 categories × (name + $ + 100 words); Foresight requires precise line-item; YC captures per-stage time+money; Brains and Astera are narrative.
  - **Milestones + go/no-go:** Google requires 3–5 dated milestones with activities (150 w) + outcomes (100 w); Foresight requires phased plan with explicit go/no-go checks; Brains B5/B6 imply this via success/failure framing; Astera Q7 frames in-scope vs. out-of-scope explicitly.
  - **Scalability + sustainability:** Google (V. Scalability + Q34.a/b); Foresight F19 (3-yr scale); YC YC26 (revenue path).
- **YC-unique:** "non-computer system you hacked" (YC2), cofounder question (YC7), regulatory path for deep-tech (YC31), 50-char tagline (YC9).
- **Astera-unique:** 300-char residency-product description (Q1), openness tradeoffs as a required section (Q8).
- **Foresight-unique:** safety-first ordering (F17), private local compute request (F34), "seminar groups of interest" (F39).
- **Google-unique:** ethics/compliance section with 6 Y/N prompts about govt entities, sanctioned countries, Google commercial ties (Q48–53); per-partner structured fields × 5 partners; "AI Maturity" self-assessment (Q27); AI force-multiplier (Q17.d).

These observations feed the harmonization in `universal_template.md`.
