# Anthropic AI for Science Program

## Eligibility
- Researchers in academia and nonprofit organizations
- Must work on scientific research topics Anthropic considers high priority (focus on biology / life sciences)
- Individual age 18+ (or age of majority in jurisdiction)
- Team / org credentials evaluated

## Funding range
Up to $20,000 in API credits. 6-month period. Credits apply only to API (NOT Claude web app). No access to non-public/non-standard models. No exemption from Anthropic Usage Policy.

## Deadline pattern
Rolling. Apply via Google Form (linked from program page).

## Submission shape
Online application form (Google Form):
- Team / project details
- Research project description
- Technical approach
- Subject-area + AI credentials of team

## Required artifacts
- Online form
- Team CVs (linked)
- Project description

## Required topics / questions
- Scientific merit
- Potential impact
- Technical feasibility
- Team credentials in (a) subject area and (b) AI
