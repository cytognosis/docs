# Astera Institute Residency (Fall 2026 Cohort)

## Eligibility
High-agency individuals: scientists, engineers, entrepreneurs. Must be able to work on-site at Astera's Emeryville, California campus. 12-18 month commitment. Project must be early-stage and well-scoped. Open to individuals globally; visa sponsorship considered.

## Funding range
- Salary $125,000 – $250,000 annually (commensurate with experience)
- Project budget $0 – $1.5M (negotiated based on project needs)
- Compute resources access
- Dedicated workspace
- Total package potentially $300k-$2M+ over 12-18 months

## Deadline pattern
Cohort-based. Fall 2026 cohort deadline: 2026-04-19. Start date 2026-08-17.

## Submission shape
Online application via Ashby HQ jobs portal. Question-driven form (per existing canonical mapping):
- Q1 (300 char): bold thesis / objective
- Q2: technical approach + budget + timeline
- Q3: team / track record / non-legible accomplishments / preliminary evidence
- Q4: state of the art and limits
- Q5: beneficiaries / users / adoption story
- Q6: why-not-academia / why-not-startup / why-Astera
- Q7: in-scope / out-of-scope (anti-goals)
- Q8: open science / outputs / licensing
- Plus: 2-min video, resume, "How did you hear" intake meta

## Required artifacts
- Online application
- Resume / CV
- 2-minute video introduction
- Track selection: Neuroscience & AI / Life Sciences & Open Science / Other
