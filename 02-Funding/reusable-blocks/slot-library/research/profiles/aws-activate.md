# AWS Activate

> **Status**: Active
> **Date**: 2026-07-10
> **Author**: @shahin
> **Audience**: leadership, grant team
> **Tags**: `funding`, `opportunity-profile`
> **Variants**: Technical (this doc) - Readable (Obsidian twin optional, same filename) - Agent (n/a)

## Eligibility
- Self-funded OR pre-Series B (most recent funding round within last 12 months)
- Fully functioning company website
- Founded within last 10 years
- Has not previously received AWS Activate Credits of equal/greater value
- Active AWS account tied to startup
- Business email matching website domain

Two paths:
- **Founders Package** (self-funded): $1,000 credits
- **Portfolio Package** (provider-backed via Activate Provider with Org ID): up to $100,000

## Funding range
- Founders: $1,000
- Portfolio: up to $100,000 USD AWS Credits
- AI startups: up to $300,000 in credits
Inventory $100,000 reflects standard Portfolio Package.

## Deadline pattern
Rolling. Decisions in 7-10 business days.

## Submission shape
Online form via aws.amazon.com/startups/credits.

## Required artifacts
- AWS account setup
- Company website
- Funding stage verification
- (Portfolio Package) Activate Provider Org ID

## Required topics / questions
- Company stage / funding
