# AWS Imagine Grant — Pathfinder GenAI

> **Status**: Active

> **Author**: @shahin

## Eligibility
US-based nonprofits with registered 501(c) status (501c3, credit unions, public utility orgs, etc). Annual budget — any size. UK/Ireland and other regional variants exist. **Pathfinder-specific**: organizations with strong existing data practice already in planning phase of AI project.

## Funding range
Pathfinder award:
- Up to $200,000 unrestricted financial support
- Up to $100,000 in AWS Promotional Credit
- Project implementation assistance
- AWS training access
- Marketing opportunities

Other Imagine Grant categories: Momentum to Modernize, Go Further Faster.

## Deadline pattern
Annual. 2026 cycle: applications open March 2026, Round 1 evaluation, Round 2 by invitation only. Inventory deadline 2026-06-06 likely Round 2 deadline.

## Submission shape
Two-round:
1. **Round 1**: open application portal — initial application
2. **Round 2**: full proposal by invitation — Pathfinder/Momentum/Go-Further-Faster track

## Required artifacts
Round 1:
- Online application
- Organizational eligibility (501c determination)
- Project summary
- Data practice assessment

Round 2 (Pathfinder):
- Full project proposal
- Technical architecture (must include frontier AI / GenAI / agentic AI as core workload)
- Budget
- Implementation plan
- Mission impact narrative
- AWS technical reference architecture / consultation outputs

## Required topics / questions
- Mission alignment
- Frontier AI use case (GenAI, agentic AI, autonomous AI as core)
- Data readiness / data practice maturity
- Technical architecture on AWS
- Scaled, repeatable solution that can impact industry/field
- Evidence of planning-phase readiness
- Implementation timeline
- Outcome metrics
