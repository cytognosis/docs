# AWS Nonprofit Credit Program

## Eligibility
501(c)(3) nonprofits in eligible countries. Verified via Goodstack / TechSoup.

## Funding range
Up to $2,000 in AWS credits annually (per inventory). Self-serve renewal.

## Submission shape
Online via AWS for Nonprofits portal + nonprofit verification.
