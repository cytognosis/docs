# EA Funds — Infrastructure Fund (EAIF)

## Eligibility
Same as LTFF: individuals, non-profit organizations, academic institutions, other entities. Global eligibility. Focuses on projects that improve the EA ecosystem (talent development, research infrastructure, fundraising, profession-specific networks).

## Funding range
Minimum $1,000. EA Funds will consider grants up to $250,000 (prefer smaller). Inventory typical $200,000. Larger grants possible for established orgs by exception.

## Deadline pattern
Rolling. Application deadlines every four months (October, February, June).

## Submission shape
Same online form as LTFF (single entry point at funds.effectivealtruism.org/apply-for-funding):
- Short Description
- Summary (most carefully reviewed)
- Detailed application
- Budget

## Required artifacts
- Online application form
- (Optional) supporting docs

## Required topics / questions
- Project goals: improving talent, capital, or knowledge access for EA-aligned work
- Theory of impact (how this strengthens EA ecosystem)
- Counterfactual (would this happen without EAIF?)
- Cost-effectiveness
- Applicant ability to execute
- Honest weaknesses
- Budget
- Timeline

## Schema gaps observed
Same as LTFF profile; both use the same application form.
- Counterfactual narrative
- Theory of impact / theory of change
- Honest weaknesses
- Individual-as-grantee eligibility

## Notes
- EAIF priorities: novel talent-development strategies, profession-specific EA networks, research infrastructure, recruiting/coordination.
- Cytognosis fit: borderline — EAIF doesn't directly fund science research (LTFF does); however, infrastructure for "open science for AI safety / global catastrophic risk" might fit.
- Same administrative process as LTFF, Animal Welfare Fund, Global Health & Development Fund.
- Same 1-2 hour application time recommendation.
