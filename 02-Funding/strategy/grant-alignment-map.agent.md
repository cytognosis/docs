# Grant-Alignment Map (agent brief)

> **Status:** Active · **Date:** 2026-07-01 · **Audience:** a fresh agent about to draft or edit a Cytognosis funding application.
> **Variants:** [technical](grant-alignment-map.technical.md) · [readable](grant-alignment-map.readable.md)

## Goal
Align every funding application to both Cytognosis direction and the funder's values, without changing deliverables and without leaking the moat.

## The one rule (framing fork)
Same work, two vocabularies, separate documents:
- **Health frame** for **Biswas** (open AI model for bipolar biotyping).
- **AI-safety frame** for **Coefficient (CDT and Capacity), LTFF, Manifund** (on-device safety layer, cognitive liberty, data capture).

## Hard constraint
**Factorized-PRS is CONFIDENTIAL (platform moat). Never name, describe, or cite it in any external application or partner doc.** If a draft references it, remove the reference.

## Three tracks (place every ask in one)
- **Cytoverse** (open map / foundation model): Biswas, AWS, NIH TMM, Templeton, IGoR TA1.
- **Cytoscope** (open sensor / UBAP): NSF X-Labs, NIH BBQS, Smart Health.
- **Cytonome / Yar** (on-device causal AI plus safety primitives): Coefficient, LTFF, Manifund, YC, One Mind.

## Five scientific threads (map each claim to one)
Dimensional biotyping; BDNF/TrkB bipolar proof-of-concept (McLean published plus Mount Sinai validation); factorized-PRS (CONFIDENTIAL, platform only); cellular micro-to-meso causal bridge (the IGoR differentiator versus HSF); on-device causal AI plus neurodivergent user research.
